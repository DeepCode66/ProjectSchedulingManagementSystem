## 功能介绍（完整见运行截图）
管理员：管理员可以通过柱状图查看各部门员工数量，以有效分配人力资源，并使用左侧菜单中的员工管理、部门管理和岗位管理功能对员工信息进行管理。

员工：员工可以通过登录界面进入系统，查询工资详情和项目调度信息，例如查看年度工资信息，查询具体月份的加班、补贴和五险一金扣款详情，以及项目调度中的请假、考勤状况。同时，员工可以在项目调度模块中编辑和删除自己的记录，并使用搜索功能筛选相关数据。
